## 🧠 ¿Qué es este repositorio?

Este repositorio sirve como **entrada principal del módulo de Big Data Aplicado**, donde se trabajan las tecnologías, conceptos y procesos fundamentales para **almacenar, procesar, analizar y visualizar grandes volúmenes de datos**, aplicándolos a casos reales.

Aquí no solo se habla de datos: **se construyen pipelines completos de datos**.

---

## 🗄️ Almacenes de Datos

Exploramos distintos enfoques de almacenamiento según la naturaleza del dato y el tipo de análisis:

### 🔹 Bases de datos NoSQL
- **MongoDB**
  - Documentos JSON
  - Esquemas flexibles
  - Ideal para datos semiestructurados
  - Escalabilidad horizontal

### 🔹 Bases de datos de Grafos
- **Neo4j**
  - Relaciones como primer ciudadano
  - Modelado de redes, comunidades y dependencias
  - Consultas con **Cypher**
  - Análisis de grafos: centralidad, comunidades, enlaces

---

## 🧩 Procesos ETL / GTL

Los datos no llegan limpios ni preparados. Aquí entra en juego el corazón del Big Data:

**ETL / GTL (Extract – Transform – Load)**

- 📥 **Extracción** de datos desde múltiples fuentes  
- 🔄 **Transformación**
  - Limpieza
  - Normalización
  - Enriquecimiento
- 📤 **Carga** en los distintos almacenes de datos

Todo el proceso orientado a **convertir datos en información útil**.
